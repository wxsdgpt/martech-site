# DESIGN.md - 设计规范与视觉系统

## 视觉定位

**核心灵感：梵高《星月夜》× AI 科技感**

将梵高后印象派的流动笔触、深邃蓝紫色调与前沿 AI 科技元素融合，
创造一种「宇宙级视野看营销科技」的沉浸感。

## 色彩系统

```
--starry-deep:    #0B1026    /* 星空深蓝（主背景） */
--starry-blue:    #1B2A4A    /* 星空蓝 */
--starry-purple:  #2D1B69    /* 星空紫 */
--swirl-cyan:     #4ECDC4    /* 漩涡青（科技感高亮） */
--swirl-gold:     #F7DC6F    /* 星光金（重点提示） */
--moonlight:      #E8E8F0    /* 月光白（正文） */

--text-secondary: #9BA4B5    /* 次要文字 */
--card-bg:        rgba(27, 42, 74, 0.6)  /* 卡片毛玻璃 */

```

## 视觉元素

### 背景
- 深蓝到深紫的渐变底色
- CSS 动画：缓慢旋转的漩涡状光带（模拟星空笔触）
- 随机闪烁的小光点（星星）
- 可选：canvas 粒子动画作为底层装饰

### 卡片
- 半透明毛玻璃效果（backdrop-filter: blur）
- 微光边框（1px 半透明青色边线）
- hover 时边框发光增强
- 圆角 16px

### 角落小插件（AI 科技感）
- **左下角**：AI 脉搏指示器 — 一个呼吸灯动画 + "AI Powered" 文字
- **右下角**：数据流动画 — 垂直滚动的代码/数据行
- **页面顶部**：粒子连线动画背景（轻量级）
- **鼠标跟随**：鼠标移动时产生微弱的星尘轨迹
- **加载动画**：星空旋涡渐显

### 排版
- 标题：思源黑体（Noto Sans SC）或系统默认，加粗
- 正文：系统字体栈，行高 1.8
- 标签：小号、圆角、渐变色背景
- 日期：月光白色，优雅衬线感

## 页面结构

### 导航
- 顶部固定导航栏，半透明毛玻璃
- 栏目：每日简报 / 深度分析 / 工具雷达 / 案例解码 / 行业报告
- 当前栏目高亮（底部发光线条）

### 内容布局
- 响应式网格，桌面 2-3 列，移动端单列
- 新闻卡片内：标题 → 摘要 → 标签行 → 来源链接
- 右侧边栏（桌面端）：AI 趋势指数、热门标签云

### 页脚
- "Powered by AI · 每日凌晨自动更新"
- 涟漪/光带动画装饰

## 交互细节

- 卡片 hover：微微上浮 + 边框发光 + 阴影加深
- 栏目切换：淡入淡出过渡
- 页面滚动：标题视差效果
- 新内容标记：金色 "NEW" 脉冲徽章

## 性能要求

- 纯前端（HTML + CSS + vanilla JS），无框架依赖
- 所有动画使用 CSS animation 或 requestAnimationFrame
- 首次加载 < 200KB（不含图片）
- 动画可降级（prefers-reduced-motion 支持）
